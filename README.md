# melhor-rastreio-teste

## Rodando o projeto
```
npm install

npm run serve
```

## Navegando no projeto

```
O projeto irá rodar em localhost:8080

A rota '/' está a parte do teste com foco em design.

A rota '/rastreio-produto' está com a parte do teste com foco no script. 

```
